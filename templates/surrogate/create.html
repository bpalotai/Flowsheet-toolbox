Surrogate Modeling Overview
What are Surrogate Models?

Surrogate models (also known as metamodels or reduced-order models) are approximations of more complex simulation models. They provide fast predictions of simulation outputs without running the full simulation.

Training Data

To create a surrogate model, you need training data. You can:

  • Use the Sampling Tool to generate and run simulation samples
  • Select existing batch simulation results
  • Upload your own training data file
Benefits
  • Significantly faster than full simulations
  • Enable rapid design space exploration
  • Support optimization and uncertainty analysis
  • Provide insights into parameter relationships
Process
  1. Generate training data from simulations
  2. Select input and output parameters
  3. Choose and train a surrogate model
  4. Validate model accuracy
  5. Use model for predictions
