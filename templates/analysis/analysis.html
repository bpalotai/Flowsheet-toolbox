Welcome to Simulation Analysis

This module allows you to perform advanced analysis on your simulation results. You can:

  • Sensitivity Analysis: Understand how input parameters affect outputs
  • Monte Carlo Analysis: Quantify uncertainty in your simulation results
  • Contribution Analysis: Identify which parameters have the most impact
  • Result Comparison: Compare multiple simulation runs side by side

To get started, select a case from the dropdown menu and choose an analysis type.
