What is Model Explainability?

Model explainability helps you understand how your simulation and surrogate models make predictions. It provides insights into which parameters have the most impact on outputs and how different features contribute to specific predictions.

Benefits
  • Build trust in model predictions and decisions
  • Identify most influential process parameters
  • Validate model behavior against domain knowledge
  • Debug and improve model performance
  • Ensure compliance with regulatory requirements
  • Gain deeper insights into process behavior
Explanation Types
Global Explanations

Understand overall model behavior across all predictions. Shows which features are generally most important and how they affect the model output on average.

Local Explanations

Explain individual predictions by showing how each feature contributed to that specific prediction. Useful for understanding model decisions case-by-case.
